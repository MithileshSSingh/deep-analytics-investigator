# Deep Analytics Investigator Agent

An agentic analytics system that diagnoses metric changes, validates likely causes, and explains what matters.

## Current Status
Initial scaffold in progress.

## MVP Goal
Support one strong investigation flow first:
- Why did revenue drop yesterday?

## Planned Routes
- Revenue investigation
- Payment failure investigation
- Refund investigation
- Funnel leak investigation

## Initial Stack
- Python
- FastAPI
- LangGraph
- DuckDB
- Pandas
- Pydantic
